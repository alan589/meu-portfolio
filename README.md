# How to run this project

### Clone this repository:

```
git clone https://github.com/alan589/meu-portfolio.git
```

### Create python venv:

On Windows:
```
py -m venv .venv
```

On Linux:
```
python3 -m venv .venv
```

### Activate venv:

On Windows:
```
.venv\Scripts\activate
```

On Linux:
```
source .venv\bin\activate
```

### Install python packages:

```
pip install -r requirements.txt
```

### Start working:

```
flask run
```
