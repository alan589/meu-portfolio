## How to run this project

Clone this repository:

```
git clone https://github.com/alan589/meu-portfolio.git
```

Create python venv:

```
py -m venv .venv
```

Activate venv
```
.venv\Scripts\activate
```

Install python packages:

```
pip install -r requirements.txt
```

Start working:

```
py -m flask run
```
